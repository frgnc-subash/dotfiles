## Installation

Just clone this repository. Either copy or symlink them as you like. Or if you dont want to clutter your home directory then clone inside the temp folder.

```
cd /tmp
git clone https://github.com/frgnc-subash/dotfiles.git
```

Go to the dotfiles folder.

```
cd dotfiles
```

Remove the .git folder

```
rm -rf .git
```

Since these are dotfiles make sure to see them using the command below.

```
ls -la
```
