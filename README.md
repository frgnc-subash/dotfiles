## Installation

Just clone this repository. Either copy or symlink them as you like. Or if you dont want to clutter your home directory then clone inside the temp folder.

```
cd /tmp
git clone https://github.com/frgnc-subash/dotfiles.git
```

Go to the dotfiles folder.

```
cd dotfiles
```

Remove the .git folder

```
rm -rf .git
```

Since these are dotfiles make sure to see them using the command below.

```
ls -la
```

## Configurations

#### After the installations configure the following.

The default folder for my wallpapers are in the following directory.

<pre>~/Pictures/wallpapers/wallpapers </pre>

You can change the directory for wallpapers in these scripts.

<pre>
~/.config/hypr/scripts/wallpaper-manager.sh
</pre>

Do similar changes for the **hyprlock** as well.

<pre> ~/.config/hypr/hyprlock.conf</pre>

Also make sure to adjust some tweaks in **hyprland config** files specially monitor resolution and refresh rate according to your system.

<pre>
~/.config/hypr/hyprland.conf
~/.config/hypr/conf/
</pre>
