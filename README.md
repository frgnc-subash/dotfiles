# 🌿 Hyprland Dotfiles

Welcome to my personal **dotfiles setup** for [Hyprland](https://github.com/hyprwm/Hyprland) — a dynamic tiling Wayland compositor. This setup is curated for performance, minimalism, and daily productivity, with carefully chosen CLI tools and aesthetic design.

## 📁 Repository Structure

```bash
.
├── hypr/           # Hyprland config
├── waybar/         # Waybar status bar configuration
├── wlogout/        # Wlogout theming
├── rofi/           # Rofi or Rofi-wayland setup
├── kitty/          # Kitty terminal config
├── wezterm/        # (Optional) WezTerm configuration
├── scripts/        # Custom scripts
├── wallpapers/     # Wallpapers used with swww
└── README.md
